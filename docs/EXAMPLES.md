# 使用示例

本文档提供一些实际的使用示例，帮助您快速上手 AI HTML Generator。

## 示例 1：个人简历页面

### 输入描述

```
创建一个现代简约风格的个人简历页面，包含：
- 顶部居中的个人照片（圆形头像）
- 姓名和职位
- 简短的个人简介
- 技能标签（如 JavaScript、Python、设计等）
- 工作经历时间线
- 教育背景
- 联系方式（邮箱、GitHub、LinkedIn）
使用柔和的蓝色作为主题色
```

### 预期输出

AI 会生成一个包含以下元素的完整 HTML 页面：
- 响应式布局
- Tailwind CSS 样式
- 优雅的渐变背景
- 卡片式设计
- 图标和动画效果

### 后续优化示例

**修改 1**：调整颜色
```
把主题色改成紫色，背景改成渐变色
```

**修改 2**：添加功能
```
在技能部分添加进度条，显示每项技能的熟练程度
```

**修改 3**：调整布局
```
把工作经历和教育背景改成左右两栏布局
```

## 实用技巧

### 1. 明确需求

✅ **好的描述**：
```
创建一个包含导航栏、Hero 区域和 3 个功能卡片的落地页，
使用蓝色渐变背景，卡片要有阴影和圆角
```

❌ **不好的描述**：
```
做一个网站
```

### 2. 分步优化

不要一次性要求所有功能，先生成基础版本，再逐步优化：

1. **第一轮**：生成基本结构
2. **第二轮**：调整样式和配色
3. **第三轮**：添加动画和交互
4. **第四轮**：优化细节

### 3. 使用具体的设计术语

- **布局**：网格布局、Flexbox、左右分栏、卡片式
- **配色**：渐变色、深色主题、玻璃态、极简风格
- **元素**：Hero 区域、CTA 按钮、时间线、手风琴
- **效果**：阴影、圆角、动画、悬停效果

### 4. 引用在线资源

可以要求 AI 使用特定的 CDN 库：

```
使用 Chart.js 显示图表
使用 AOS.js 添加滚动动画
使用 Animate.css 添加进入动画
从 Unsplash 使用图片
```

### 5. 指定响应式需求

```
确保在手机端也能良好显示，小屏幕时改为单列布局
```

## 常见问题解决

### Q: 生成的 HTML 没有样式？

**A**: 确保 AI 生成的代码包含了 Tailwind CSS CDN：
```html
<script src="https://cdn.tailwindcss.com"></script>
```

如果没有，在对话中说：
```
请添加 Tailwind CSS CDN 引用
```

### Q: 图片显示不出来？

**A**: 要求 AI 使用在线图片服务：
```
使用 Unsplash 的图片链接，格式：
https://images.unsplash.com/photo-xxxxx
```

### Q: 想要添加 JavaScript 交互？

**A**: 明确说明需要的交互效果：
```
添加一个手风琴效果，点击标题展开/收起内容
```

### Q: 样式不满意？

**A**: 具体描述想要的改动：
```
把卡片的圆角改大一些，阴影改淡一些，
背景色从蓝色改成紫色渐变
```

### Q: 生成的 HTML 太大？

**A**: 要求优化：
```
请简化代码，移除注释和多余的空格，
把 base64 图片改成外部链接
```
